# Trabalho-Processamento-de-Sinais-I-1
CEFET/RJ
Disciplina: Processamento de Sinais I

Autores: Caio Bernardo, Fabio Daniel dos Santos Lacerda, Leonardo Sant’Ana Bittencourt.

Este repositório contém a implementação e análise desenvolvidas na Aula Prática 1 da disciplina de Processamento de Sinais.
- O projeto aborda conceitos fundamentais como:
- Geração de sinais cossenoidais
- Sinais do tipo chirp
- Manipulação e reprodução de áudio
- Resposta ao impulso de sistemas acústicos
- Convolução entre sinais

Objetivos:
- Compreender sinais no domínio do tempo
- Explorar a relação entre frequência e percepção auditiva
- Analisar sinais reais de áudio
- Entender o comportamento de sistemas lineares invariantes no tempo (LTI)
- Aplicar convolução para simular ambientes acústicos

Tecnologias Utilizadas:
- Google Colab
- Python
- NumPy
- SciPy
- Matplotlib
- IPython Display

Referências:
- Material da disciplina
- Documentação SciPy / NumPy
- Conteúdo teórico de Processamento de Sinais
